# User Flow

## 1. 主要用户路径

```text
用户打开网站
  -> 浏览今日 AI 创意情报
  -> 查看 Today's Signal
  -> 按分类筛选
  -> 点击 Signal Card
  -> 查看创意转译详情
  -> 获得项目 / 商业 / Prompt 灵感
  -> 跳转原文
```

## 2. Home 浏览流程

1. 用户进入首页。
2. 页面显示 `AI Creative Radar` 和副标题 `Visual AI signals for creators`。
3. 用户看到今日日期和 Today's Signal。
4. 页面默认展示全部 Signal Card。
5. 用户扫读每张卡片的分类、标题、趋势判断、适合人群、相关工具和视觉标签。
6. 用户点击分类筛选，只看 AI视觉工具、创意案例、潮流趋势、商业玩法、灵感Prompt 或前沿观察。
7. 用户点击感兴趣的 Signal Card 进入详情。

## 3. Signal Detail 流程

1. 用户从 Signal Card 进入详情。
2. 页面展示标题和一句话趋势判断。
3. 用户阅读“发生了什么”，理解资讯背景。
4. 用户阅读“对视觉创作者有什么用”，理解可用价值。
5. 用户查看“可以做成什么项目”，获得作品集、内容选题或实验方向。
6. 用户查看“商业化可能”，判断是否适合接单、副业、内容账号或小产品。
7. 用户查看 Prompt 灵感和相关工具。
8. 用户点击原文链接查看来源。
9. 用户返回首页继续浏览。

## 4. About 流程

1. 用户点击 About。
2. 页面说明产品定位和适合人群。
3. 页面说明为什么做这个项目。
4. 页面介绍项目作者。
5. 页面说明如何帮助视觉创作者把 AI 信息变成作品和机会。

## 5. 第一版不处理的流程

- 用户注册或登录。
- 用户收藏 Signal。
- 用户评论或投稿。
- 邮件订阅。
- 个人推荐算法。
- 后台录入或审核系统。
